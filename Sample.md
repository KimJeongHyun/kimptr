+ **게임소개**
  1. 시나리오
        + 낮
            1. 시민 및 마피아의 토론 및 미션 수행
            2. 토론 결과를 통해 사형 집행
        + 밤
            1. 시민들은 잠에 빠짐
            2. 마피아는 전략 토론 및 npc 또는 유저 살해
        + 낮과 밤의 반복
            1. 한 팀의 승리조건이 충족될 때 까지?
            2. 한 팀이 다 남을 때까지
  2. 게임 기본 구성
        + 총 수행 인원
            - 8명
            - 시민 팀, 마피아 팀으로 분배
            - 각 직업은 랜덤하게 설정
        + 시민 팀
            1. 시민
                - 무능력자
            2. 군인
                - 지목 당할 시 1턴 방어 가능
                - 시민이 모두 죽으면 실패
            3. 경찰
                - 인원 지목 시 시민/마피아 판별
                - 직업을 밝힐 수는 없음
            4. 의사
                - 인원 지목 시 해당 인원 소생
                - 소생되는 대상은 마피아의 타겟이어야함
                - 지목되는 대상은 
        + 마피아 팀
            1. 마피아
                - 유저 및 조건부 npc 살해 가능
                - '마피아' 끼리 파악 가능
            2. 스파이
                - 시민 팀에 섞인 X맨
                - 스파이는 마피아의 신원 파악
                - 마피아는 스파이 신원 파악 불가능
                - 마피아 전략 토의 참여 불가능
            3. 늑대인간
                - 초기엔 일반 시민
                - 마피아와 접선 시 마피아 팀 소속으로 변경
                    1. 마피아가 살해 타겟으로 지정한 경우
                    2. 선택한 사람이 마피아에게 살해됐을 경우
                    3. 마피아 전략 토의에 참여 가능
                - 플레이어는 접선 전 본인의 늑대인간 여부를 알 수 없음
        + NPC
            1. 시민에게 힌트 부여
                - NPC 사망시 힌트 습득 불가능
            2. 마피아 미션 성공 시 사망 가능
            3. 생성위치는 게임에 따라 랜덤
        + 미션
            1. 시민
                - npc가 부여
                - 형태
                    1. 게임 내 장치(NPC, 아이템 등)를 통한 수행
                    2. 실패 시 해당 힌트 습득 영구불가능
                - 달성 시 선택
                    1. 마피아 힌트 부여
            2. 마피아
                - 시스템이 자동 부여
                - 형태
                    1. 특정 대상에 대한 행동
                    2. 특정 아이템 사용
                - 달성 시 선택 
                    1. npc 암살
                    2. 아이템 획득
        + 제한 시간
            1. 낮
                - 5분 내외
            2. 밤
                - 2분 내외
            3. 게임 전체 시간
                - 30분 소요 예정
        + 승리 & 패배 조건
            1. 시민 팀
                - 승리조건
                    1. 마피아 전멸
                - 패배조건
                    1. 시민 전멸
            2. 마피아, 스파이
                - 승리조건
                    1. 시민 전멸
                - 패배조건
                    1. 마피아 전멸
            3. 늑대인간
                - 승리조건
                    1. 시민 팀인 경우
                        - 마피아 전멸
                    2. 마피아 
                        - 시민 전멸
        + 레이팅 제도
            1. mmr 도입
                - 단순 수치화
                - 실력의 증명
                - 매칭과 연관 없음
            2. 보상
                - 특정 수치 도달 시
                    1. 게임 재화 제공
                    2. 특수 스킨 또는 캐릭터 제공
                        - 밸런스와 
        + 음성 채팅
            1. 입력
                - 핸드폰 마이크
                - 이어폰 마이크
            2. 출력
                - 이어폰 스피커
                - 핸드폰 스피커
            3. 특징
                - 인 게임 활용
                    1. 일반적 경우
                        - 같은 방 안의 유저와만 음성 채팅 가능
                        - 유저의 거리에 따라 음성 Fade in, Fade out 기능
                    2. 특수한 경우
                        - 능력의 사용
                            1. 전체 대상 음성 채팅
                        - 마피아
                            1. 마피아끼리 음성 채팅 on/off 가능
         + 게임 재화
            1. 게임 머니
                - 획득 경로
                    1. 게임 승리 또는 패배 시 획득 가능
                - 사용처

                - 사용처
                    1. 아이템 구매

                    3. 스킨 구매
          + 매칭 시스템
            1. 솔로 플레이
                - 솔로 플레이어들과만 매칭
            2. 다중 플레이
                - 파티 기능
                    1. 친구와 함께 파티 가능
